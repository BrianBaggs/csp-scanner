csp-scanner is a python script for scanning a Content Security Policy for weak directives that can extend the attack surface and allow attackers to exploit your website or web application.
Findings are ordered by severity and by default, the results are outputted in JSON, HTML, and TXT.

Python 3 stdlib only — no pip dependencies required.
